== Description ==

**Woo Product Showcase** – WooCommerce Product List Widget and Shortcodes | WooCommerce Product Grid Widget and Shortcodes | WooCommerce Category and Product List Widget and Shortcodes. The plugin facilitate multiple WooCommerce product widgets and shortcodes in grid and list view with the selected product category.


### The Plugin Features:
* **New - AFT Product Grid Shortcode** - Display WooCommerce products in a grid format.
`[woo-product-showcase mode='grid' number_of_products='6' product_cat='0' show='featured' orderby='date' order='desc']`
* **New - AFT Product List Shortcode** - Display WooCommerce products in a list format.
`[woo-product-showcase mode='list' number_of_products='6' product_cat='0' show='featured' orderby='date' order='desc']`
* **New - AFT Product Express Grid Shortcode** - Display WooCommerce products in a category images and details along with product grid format.
`[woo-product-showcase mode='express-grid' number_of_products='6' product_cat='0' show='featured' orderby='date' order='desc']`
* **AFT Product Grid Widget** - Display WooCommerce products in a grid format.
* **AFT Product List Widget** - Display WooCommerce products in a list format.
* **AFT Product Express Grid Widget** - Display WooCommerce products in a grid format with a product category thumbs and descriptions.


== Installation ==
* Download to your plugin directory or simply install via WordPress admin interface.
* Activate.
* Go to Appearance>Widgets.
* Place widgets to your desired section.
* Set appropriate setting.
* Save.
